# Schema for Curated Dataset

| Column | Type | Description |
|---------|------|--------------|
| Name | Text | Meteorite name |
| ID | Whole Number | NASA record ID |
| NameType | Text | Type of name |
| Class | Text | Meteorite classification |
| Mass_g | Decimal Number | Mass in grams |
| Fall | Text | Fell / Found |
| Year | Whole Number | Year of event |
| Latitude | Decimal Number | Latitude |
| Longitude | Decimal Number | Longitude |
| Source_URL | Text | Provenance link to NASA dataset |
